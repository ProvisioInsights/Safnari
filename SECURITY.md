# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue, please report it to
`hello@provisioinsights.com`.

We aim to respond within **72 hours**.

## Supported Versions

We support the **latest release only**.

## Disclosure

This is an open-source project with no bug bounty program. Responsible
disclosure is appreciated, and contributors are welcome to suggest fixes via
GitHub (issues or pull requests) after first reporting the issue.
